# RetailIQ: Customer & Sales Intelligence Dashboard

## Objective
Develop an end-to-end business intelligence solution using Python, SQL, Power BI, and Excel to analyze sales performance, customer behavior, profitability, and business trends.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- Excel

## Dataset
Superstore Sales Dataset (9,994 records)

## Key Business Questions
1. What is total revenue and profit?
2. Which regions generate the highest sales?
3. Who are the top customers?
4. Which products are most profitable?
5. What factors contribute to losses?
6. What are future sales projections?

## Project Workflow
Data Collection
→ Data Cleaning
→ SQL Database Design
→ Exploratory Data Analysis
→ Business Insights
→ Dashboard Development
→ Forecasting
